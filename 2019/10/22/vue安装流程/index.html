vue安装流程

发表于 2019-10-22 | 分类于 Vue的使用
字数统计: | 阅读时长 ≈

1、安装node.js

检查是否成功：node-v npm-v

2、全局安装脚手架工具vue-cli，命令如下：

npm install –global vue-cli

3、检查是否安装成功

vue -V打印版本号

4、创建项目

若没有安装webpack，则先安装webpack
使用npm install -g webpack命令
创建项目
使用 vue init webpack myVue命令
myVue：项目名称

5、安装配置编写

项目名称–>项目描述–>项目作者–>是否安装路由vue-router–>是否启用eslint检查
注：安装过程 中有个选项（Use ESLint to line your code ?选择 No ）

6、cd进入到myVue目录下，使用npm install 安装package.json包中的依赖

命令如下：

cd myVue

npm install

7、运行项目

cd进入项目文件夹–使用npm run dev命令运行
localhost:8080访问 将会出现vue初始页面：
如果出现错误，端口被占用错误，修改端口
修改ip地址或端口号
config—>index.js dev–>host:”ip地址” port:”0000”

项目制作开发完毕后进行打包
通过命令npm run build 进行打包
注意：在打包之前修改config下index.js中的
build:{
assetsPublicPath:”./“
}

在脚手架中.vue的文件就是一个组件
